Speechify TTS stream plus Web Audio API

Play PCM as it arrives

The browser posts to a local /v1/audio/stream proxy. The server adds the API key, asks Speechify for audio/pcm, and this page queues each 16-bit chunk into an AudioContext.

This demo intentionally avoids decodeAudioData. That API expects a complete encoded file. Raw 24 kHz mono PCM can be converted directly from signed 16-bit samples to Float32 samples, then scheduled back to back with source.start(playhead).
